# Transformer

This code implements a single Transformer block from scratch in NumPy: scaled dot-product self-attention, a two-layer feed-forward network with a ReLU nonlinearity, and layer normalization wired together with residual connections. The block computes `LayerNorm(X + Attention(X))` followed by `LayerNorm(X + FeedForward(X))`, which is the core building unit stacked repeatedly in real Transformer models.

Building this clarified what each piece actually does underneath the framework abstractions. Attention is a similarity score between every pair of tokens (`QKᵀ`), normalized by softmax and used to mix value vectors — scaling by `√d_k` keeps softmax from saturating. The feed-forward network expands each token into a wider hidden space and projects it back, letting the model transform the representation independently per position. Layer norm rescales each token's features to zero mean and unit variance so the residual stream stays numerically stable as blocks compose, and the residual connections themselves are what let gradients flow through deep stacks.

The output is a matrix where each row is an updated representation of the corresponding input token — enriched with context from the rest of the sequence via attention and then transformed nonlinearly by the feed-forward layer. Running the script prints the result of passing a random 4-token, 8-dimensional input through one such block.
